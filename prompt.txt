🔥 Nama Proyek

Smart Inspector Scrapper — Chrome/Edge Extension untuk scraping elemen spesifik pada website melalui mode inspeksi visual atau selector otomatis.

🎯 Tujuan Utama

Menyediakan alat scraping berbasis browser yang memudahkan pengguna mengambil konten dari website tanpa coding dengan dua metode:

Mode Manual (Inspector Mode) → klik elemen untuk mengambil datanya

Mode Otomatis (Auto Selector Mode) → menentukan selector dan scraping seluruh elemen yang cocok secara massal

🌟 Fitur Utama
Fitur	Deskripsi
Inspector Mode	User mengarahkan & mengklik elemen untuk scraping
Auto Selector Mode	User menentukan selector (class/id/custom) → scrape semua elemen
Penamaan Selector	Tiap selector dapat diberi nama (contoh: harga, judul, gambar)
Preview Data	Tampilkan hasil scraping dalam tabel di popup
Export Data	JSON / CSV / copy clipboard
Template Scraping	Simpan konfigurasi selector untuk scraping ulang di website yang sama
Highlight saat hover	Elemen di-highlight saat mode inspeksi aktif
Ambil elemen detail	text, html, attributes (href, src, alt, data-*)
Pagination (opsional)	Fitur klik tombol “Next Page” otomatis untuk crawling multi halaman
🧠 Data yang diambil dari elemen

Saat scraping (manual atau otomatis) extension mengumpulkan:

selector (CSS)

xpath (opsional)

innerText

innerHTML

element attributes → href, src, alt, dll

outerHTML (opsional)

custom_name yang diberikan user

🧩 Struktur penyimpanan selector

Format JSON internal untuk selector:

[
  {
    "name": "judul",
    "selector": ".product-card .title",
    "type": "text"
  },
  {
    "name": "harga",
    "selector": ".product-card .price",
    "type": "text"
  },
  {
    "name": "gambar",
    "selector": ".product-card img",
    "type": "attribute",
    "attr": "src"
  }
]

🔄 Alur Mode Manual (Inspector)

User mengaktifkan Inspector

Elemen yang dilewati mouse highlight

User klik elemen

Muncul popup kecil untuk mengisi:

Nama selector (contoh: judul_halaman)

Data hasil klik dikirim ke popup extension

User bisa export / simpan selector

🔁 Alur Mode Otomatis (Auto Selector)

User input selector + nama (contoh: .item .price → harga)

Klik Scrape Now

Extension iterasi seluruh elemen cocok selector

Hasil scraping berupa array objek:

[
  { "judul": "Produk 1", "harga": "20.000", "gambar": "link1" },
  { "judul": "Produk 2", "harga": "25.000", "gambar": "link2" }
]
